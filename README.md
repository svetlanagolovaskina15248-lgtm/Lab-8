🧪 Lab 8 — Работа с OpenCV

В лабораторной работе реализованы программы для обработки изображений и видеопотока с использованием OpenCV.

🔹 Содержание

* Преобразование изображения — перевод в полутоновый формат (grayscale).
* Отслеживание метки — поиск изображения на видеопотоке с камеры.
* Координаты метки — вывод положения найденной метки на экране.
* Наложение изображения — отображение мухи в центре найденной метки.

⚙️ Технологии

Python, OpenCV, NumPy
